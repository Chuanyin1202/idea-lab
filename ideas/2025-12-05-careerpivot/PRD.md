# 產品名稱
CareerPivot

# One-line Pitch
CareerPivot 是一款個性化的職業策略應用，幫助專業人士制定量身定制的職業轉型計劃。

# Background & Problem Statement
隨著職場壓力的增加，許多專業人士感到在當前角色中陷入困境，無法找到適合自己的職業道路。傳統的職業建議往往過於通用，無法考慮個人具體情況，如技能、財務狀況和家庭約束。CareerPivot 將這些因素納入考量，提供個性化的職業轉型計劃。

# Target Users & Personas
目標用戶為中年專業人士，年齡在30至50歲之間，通常在職場中感到疲憊或無法實現職業目標。他們可能面臨家庭責任、財務壓力或對未來職業發展的困惑。

## Personas
1. **職場倦怠者**：在同一工作崗位上工作多年，感到無法再成長。
2. **轉型求職者**：希望轉換職業，但不確定如何開始。
3. **家庭負擔者**：需要考慮家庭責任和財務狀況，尋求可行的轉型方案。

# User Pain Points
- 缺乏針對個人情況的具體職業建議。
- 不知道如何評估自己的可轉移技能。
- 在轉型期間擔心財務安全。
- 難以找到支持和指導。

# Value Proposition
CareerPivot 提供量身定制的職業轉型計劃，考慮用戶的具體情況，幫助他們找到合適的職業路徑，並提供必要的資源和支持，減少轉型過程中的焦慮和不確定性。

# Core Use Cases & User Stories
1. **制定職業轉型計劃**：用戶輸入個人信息，系統生成 6 個月、1 年和 2 年的職業轉型計劃。
2. **技能評估**：用戶可以評估自己的技能並獲得可轉移技能的建議。
3. **財務計算**：系統計算轉型期間所需的薪水橋接。
4. **社群連接**：用戶可以連接到成功轉型的專業人士，獲取經驗分享。

# MVP Scope
## Must-have
- 用戶登錄和註冊功能
- 個性化職業轉型計劃生成器
- 可轉移技能評估工具
- 薪水橋接計算器
- 社群連接平台

## Nice-to-have
- 每月檢查的過渡教練
- 薪資談判腳本
- 面試準備資源

# User Flow Overview
1. 用戶註冊並填寫個人信息。
2. 系統分析信息並生成職業轉型計劃。
3. 用戶獲得可轉移技能建議和薪水橋接計算。
4. 用戶連接到社群，獲取支持。

# Basic System Architecture
- **前端**：React.js
- **後端**：Node.js + Express
- **數據庫**：MongoDB
- **雲服務**：AWS 或 Google Cloud
- **身份驗證**：OAuth 2.0

# Monetization Hypothesis
- 每月收費 $49 的高級訂閱計劃，提供額外的資源和支持。
- 企業健康計劃的年度收費在 $2,000 至 $10,000 之間。

# Success Metrics
- 每月活躍用戶數 (MAU)
- 訂閱轉化率
- 用戶滿意度評分
- 成功轉型的用戶數量

# Risks & Assumptions
- 用戶對個性化建議的需求是否足夠強烈。
- 競爭對手的反應及市場接受度。
- 技術實現的可行性和成本控制。
