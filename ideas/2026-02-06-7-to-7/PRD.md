# 產品名稱
7 to 7

# One-line Pitch
為夜班工作者量身打造的睡眠訓練應用，幫助他們在不穩定的工作時間中獲得更好的睡眠質量。

# Background & Problem Statement
在美國，有超過兩千萬人從事夜班工作，這些工作者的作息時間經常變動，導致他們面臨嚴重的睡眠問題。現有的睡眠應用大多假設用戶有固定的就寢時間，對於這些夜班工作者來說，這些建議往往無法實用。缺乏針對性和個性化的睡眠管理工具，使得這些工作者在面對工作與健康的挑戰時，難以找到有效的解決方案。

# Target Users & Personas
- **主要用戶**: 夜班工作者，包括護士、醫護人員、倉庫工人、工廠工人及警察等。
- **用戶角色**:
  - **護士**: 需要在不同的班次之間快速調整自己的生理時鐘，保持警覺和健康。
  - **倉庫工人**: 在輪班工作中，經常面臨疲勞和睡眠不足的挑戰。
  - **警察**: 需要在不規則的工作時間中保持高效的反應能力和精神狀態。

# User Pain Points
- 難以在不規則的工作時間中獲得充足的睡眠。
- 缺乏針對夜班工作者的睡眠調整建議。
- 目前的睡眠應用無法考慮到用戶的工作輪班安排。
- 長期的睡眠不足導致健康問題和工作效率下降。

# Value Proposition
7 to 7 提供一個專為夜班工作者設計的睡眠訓練應用，通過個性化的睡眠計劃和實時調整，幫助用戶有效管理他們的睡眠質量，減少疲勞感，提升工作效率和整體健康。

# Core Use Cases & User Stories
- **用例1**: 用戶輸入下一周的工作排班，應用生成個性化的睡眠轉換計劃。
  - **用戶故事**: 作為一名護士，我希望能夠根據我的工作班次獲得調整建議，以便能夠更好地適應新的作息時間。
  
- **用例2**: 應用根據用戶的疲勞得分調整建議。
  - **用戶故事**: 作為一名夜班工人，我希望能夠記錄我的疲勞感，並獲得相應的建議來改善我的睡眠質量。

# MVP Scope (Must-have vs Nice-to-have)
- **Must-have**:
  - 用戶排班輸入功能
  - 個性化睡眠計劃生成
  - 疲勞得分記錄和分析
  - 與Apple Watch或Fitbit的整合

- **Nice-to-have**:
  - 社群功能，讓用戶分享經驗
  - 數據可視化儀表板，顯示睡眠質量趨勢
  - 進階的睡眠分析報告

# User Flow Overview
1. 用戶註冊和登入
2. 輸入工作排班和通常的睡眠時間
3. 應用生成個性化的睡眠計劃
4. 用戶記錄疲勞得分
5. 應用根據數據調整建議

# Basic System Architecture
- **前端**: iOS應用，使用Swift開發
- **後端**: Node.js + Express
- **數據庫**: Firebase
- **健康數據整合**: HealthKit
- **算法**: 使用Dimesimeter研究進行生理時鐘計算

# Monetization Hypothesis
- 個人用戶訂閱制：每月$9.99
- B2B模式：醫療機構可支付$10,000-$50,000的年費，獲取部門級儀表板和數據分析。

# Success Metrics
- 訂閱用戶數量
- 用戶留存率
- 用戶疲勞得分的改善幅度
- B2B客戶的滿意度和續約率

# Risks & Assumptions
- **風險**: 用戶對於數據隱私的擔憂可能影響註冊率。
- **假設**: 夜班工作者願意為改善睡眠質量支付訂閱費用。
