# 產品名稱
Found And Tested

# One-line Pitch
一個經過技能測試的虛擬助手市場，專為小型企業設計。

# Background & Problem Statement
在當前的自由職業平台上，小型企業主面對大量的虛擬助手（VA）提案，但這些提案常常缺乏差異化，導致選擇困難。小型企業主通常需要花費大量時間來篩選合適的VA，這與他們雇用VA的初衷相悖。Found And Tested旨在解決這一問題，通過預先篩選和測試VA的技能，幫助企業主快速找到合適的人選。

# Target Users & Personas
- **小型企業主**：需要虛擬助手來處理行政工作、客戶服務或社交媒體管理。
- **虛擬助手**：希望在平台上獲得更多的工作機會，並提升自己的市場競爭力。

# User Pain Points
- 雇用虛擬助手的過程繁瑣，需花費大量時間篩選。
- 難以評估VA的真實技能和過往表現。
- 缺乏可靠的篩選機制，導致不合適的雇用決策。

# Value Proposition
Found And Tested提供一個經過嚴格篩選的虛擬助手市場，讓小型企業主能夠快速找到合適的VA，並通過數據驅動的匹配系統提高雇用效率。

# Core Use Cases & User Stories
1. **小型企業主登錄平台**：企業主可以快速發布工作需求，並查看經過篩選的VA名單。
2. **VA提交申請**：虛擬助手可以在平台上提交申請，並通過技能測試和背景檢查。
3. **數據驅動的匹配**：系統根據企業主的需求和VA的技能進行匹配，提供最佳候選人選擇。

# MVP Scope (Must-have vs Nice-to-have)
- **Must-have**:
  - VA技能測試系統
  - 背景檢查功能
  - 基本的匹配算法
  - 用戶註冊和登錄功能
- **Nice-to-have**:
  - 高級的數據分析和報告功能
  - 用戶評價系統
  - 移動應用版本

# User Flow Overview
1. 小型企業主註冊並登錄。
2. 發布工作需求。
3. VA註冊並通過技能測試。
4. 系統根據需求和技能進行匹配。
5. 企業主查看匹配結果並選擇VA。

# Basic System Architecture
- **前端**：React.js
- **後端**：Node.js + Express
- **數據庫**：MongoDB
- **雲服務**：AWS（用於存儲和計算）
- **第三方服務**：Stripe（支付處理）

# Monetization Hypothesis
- 企業主每月支付49至149美元的訂閱費用，並收取10%的安置費。
- VA每月支付29美元以獲得高級位置和驗證徽章。

# Success Metrics
- 每月活躍用戶數（MAU）
- 每月新增VA數量
- 平均時間到雇用（Time-to-Hire）
- 用戶滿意度評分

# Risks & Assumptions
- 風險：市場競爭激烈，可能面臨其他平台的挑戰。
- 假設：小型企業主願意為篩選過的VA支付額外費用以節省時間。
