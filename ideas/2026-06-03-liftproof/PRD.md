# 產品名稱
Liftproof

# One-line Pitch
一款基於手機的運動評估應用，提供快速、便宜的工前體檢，幫助企業降低工傷風險。

# Background & Problem Statement
在工業招聘中，應聘者通常需要進行繁瑣的體檢流程，這不僅耗時，還可能導致候選人因等待而錯失工作機會。傳統的臨床體檢費用高達200至400美元，並且需要花費2小時的時間。這對於需要迅速招聘的企業來說，既不高效也不經濟。根據統計，因工傷造成的損失每年高達數十億美元，這表明需要一種更有效的解決方案來評估應聘者的身體狀況。

# Target Users & Personas
- **招聘經理**：需要快速評估應聘者的身體能力，降低工傷風險。
- **應聘者**：希望快速完成招聘流程，獲得工作機會。
- **企業HR**：希望減少招聘成本，提高招聘效率。

# User Pain Points
- 傳統體檢耗時且費用高昂。
- 應聘者在等待體檢結果期間無法開始工作。
- 招聘經理難以快速獲得應聘者的身體狀況評估。

# Value Proposition
Liftproof 提供一個簡單、快速且便宜的運動評估解決方案，讓應聘者可以在家中完成體檢，並在24小時內獲得結果，從而加快招聘流程並降低企業的工傷風險。

# Core Use Cases & User Stories
1. **應聘者使用應用進行自我評估**：
   - 作為應聘者，我希望能在手機上完成運動評估，以便快速獲得工作機會。
2. **招聘經理查看評估報告**：
   - 作為招聘經理，我希望能在24小時內收到應聘者的評估報告，以便快速做出招聘決策。
3. **企業HR管理評估數據**：
   - 作為企業HR，我希望能夠管理所有應聘者的評估數據，以便進行後續的分析和決策。

# MVP Scope (Must-have vs Nice-to-have)
## Must-have
- 手機應用程序，支持運動評估（蹲下、伸展、舉起模擬、平衡保持）。
- AI 評分系統，根據運動範圍、舉起機制和受傷風險標記生成報告。
- 24小時內生成的通過/不通過報告。

## Nice-to-have
- 數據分析儀表板，供企業HR查看評估趨勢。
- 應用內即時聊天功能，供應聘者與招聘經理溝通。

# User Flow Overview
1. 應聘者下載並安裝 Liftproof 應用。
2. 應聘者註冊並登錄。
3. 應聘者按照應用指示進行運動評估。
4. AI 系統分析數據並生成報告。
5. 招聘經理收到報告並做出招聘決策。

# Basic System Architecture
- **前端**：React Native (手機應用)
- **後端**：Node.js + Express
- **數據庫**：MongoDB
- **AI 模型**：Python + TensorFlow
- **雲服務**：AWS (存儲和計算)

# Monetization Hypothesis
每位應聘者的運動評估收費30至50美元，預計每月可吸引1000名應聘者，年收入可達36萬至60萬美元。

# Success Metrics
- 每月活躍用戶數 (MAU)
- 應聘者評估完成率
- 招聘經理的滿意度評分
- 年收入增長率

# Risks & Assumptions
- **風險**：應聘者可能對使用手機應用進行體檢的信任度不足。
- **假設**：企業會願意接受這種新的體檢方式，並且能夠看到其帶來的成本效益。
