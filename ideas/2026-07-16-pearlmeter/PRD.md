# 產品名稱
Pearlmeter

# One-line Pitch
透過手機相機掃描牙齒顏色，幫助用戶追蹤牙齒美白進度，避免浪費金錢在無效產品上。

# Background & Problem Statement
在家美白牙齒的市場規模達到75億美元，然而消費者在選擇產品時往往依賴主觀判斷，這導致了大量的金錢浪費。許多用戶在嘗試不同的美白產品時，平均花費高達300美元，卻無法確保效果。牙醫會使用校準的色卡來評估牙齒顏色，但普通消費者只能依賴鏡子中的模糊反映。Pearlmeter旨在提供一個科學且可視化的解決方案，幫助用戶準確追蹤牙齒的美白進度。

# Target Users & Personas
1. **年輕女性**：關注外貌，經常使用牙齒美白產品，願意嘗試新科技。
2. **中年男性**：注重形象，尋求有效的牙齒美白解決方案，對產品效果有高期望。
3. **牙科患者**：剛接受牙齒治療的患者，希望在家中維持牙齒的美白效果。

# User Pain Points
- 無法準確評估牙齒顏色的變化。
- 在選擇美白產品時缺乏科學依據。
- 花費大量金錢在效果不佳的產品上。
- 需要一個簡單易用的工具來記錄和追蹤進度。

# Value Proposition
Pearlmeter將手機變成一個牙齒顏色掃描儀，提供準確的牙齒顏色評估，並根據用戶的牙齒顏色變化提供個性化的產品建議，幫助用戶最大化美白效果，減少不必要的開支。

# Core Use Cases & User Stories
1. **用戶掃描牙齒顏色**：用戶在良好光線下拍攝牙齒照片，應用程序分析顏色並給出評分。
2. **追蹤進度**：用戶每周拍攝牙齒照片，應用程序記錄顏色變化並顯示進度圖表。
3. **產品推薦**：根據用戶的牙齒顏色變化，提供有效的美白產品建議。

# MVP Scope (Must-have vs Nice-to-have)
## Must-have
- 手機相機掃描牙齒顏色的功能。
- 顏色評分系統，基於牙科色卡。
- 進度追蹤圖表。
- 產品推薦系統。

## Nice-to-have
- 社交功能，讓用戶分享進度。
- 教學視頻，指導用戶如何使用美白產品。
- 數據分析報告，顯示市場上最有效的產品。

# User Flow Overview
1. 用戶下載並安裝Pearlmeter應用程序。
2. 用戶創建帳戶並填寫牙齒美白歷史。
3. 用戶在良好光線下拍攝牙齒照片。
4. 應用程序分析照片並顯示顏色評分。
5. 用戶定期拍攝照片以追蹤進度。
6. 應用程序根據顏色變化提供產品建議。

# Basic System Architecture
- **前端**：React Native（移動應用開發）
- **後端**：Node.js + Express（API服務）
- **數據庫**：MongoDB（用戶數據和產品信息存儲）
- **影像處理**：OpenCV（牙齒顏色分析）

# Monetization Hypothesis
- 提供免費基本功能，收取高級功能的訂閱費用。
- 與美白產品品牌合作，通過推薦獲取佣金。
- 在應用內銷售個性化的美白產品。

# Success Metrics
- 用戶增長率：每月新增用戶數量。
- 用戶留存率：使用應用程序超過三個月的用戶比例。
- 產品推薦的轉換率：通過應用程序推薦的產品銷售數量。

# Risks & Assumptions
- 用戶對於使用手機相機進行牙齒顏色評估的準確性存疑。
- 市場競爭激烈，需確保產品的獨特性和價值。
- 需要持續更新產品推薦以保持用戶的興趣。
